🌱 Roadmap Plateforme Anti-Gaspillage

⏱️ 1-2 semaines

Technologies

Figma Adobe XD Node.js React PostgreSQL Swagger

Node.js Express PostgreSQL JWT Stripe API Postman

Technologies

React CSS Tailwind Axios Leaflet.js Google Maps
